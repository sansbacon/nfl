Scrambled Data

Scrambled data means the data values are adjusted up or down by about 20% from their actual value. Certain types of data are scrambled in our Free Trial and Developer Subscription, such as team & player stats and fantasy stats. All other data, such as teams, players, schedules, etc, including their unique IDs, are not scrambled and the same as you would see in our Post-Game, Live and Real-Time Subscriptions.

This is done so that you can use the Free Trial (1,000 API calls/mo) or Developer Subscription (unlimited API calls, $29/mo) to complete the majority of your integration before having to upgrade to one of our premium API subscriptions.
